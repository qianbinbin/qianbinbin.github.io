生命没有意义，“意义”是由人自己来定义的。

很久以前就得出这样的结论。不知道为何而活，又缺乏自杀的勇气，何况那么多人在恬不知耻地活着。如果在我有自我意识前可以选择，那么我会选择从未诞生。

后来才知道，原来这在哲学上叫虚无主义。

也问过同学，她似乎也有类似的观点，但又补充说，不妨把生命当作一种馈赠。

可惜我做不到这么豁达。它更像是一种负担，或者更乐观一点，一种责任。

如果生命是一个程序，那么它或许类似于：

```
class Life {
    void findMeaningOfLife() {
        Life child;
        while ((child = new Life()) != null)
            child.findMeaningOfLife();
    }
}
```

这是一个无限的递归，不同的是，栈不会那么容易就溢出。

有趣的是，在寻找生命意义的函数体中，仅剩的是“繁衍”。

“繁衍”，不是经常被当作生命的意义吗？尤其是在古代，生产力低下的社会，多一个人口就多一个劳动力。为人父母不需要征求孩子的意见，就可以决定孩子的出生。其实现代也好不到哪去，因为连创造生产力这一借口都显得卑鄙。

至于这一程序最初的调用者是谁？为什么要演化出生命？

不知道。

《道德经》说：天地不仁，以万物为刍狗。意思是天地无所谓仁爱，任由万物自生自灭。

如果“天地”是某种超越一切的意识，那么它应该是一个意大利面怪物，在一次严重的酗酒之后创造了宇宙。

但是按照现有物理学的观点，据说宇宙产生于一次大爆炸，在那之前，没有意义。大爆炸是用来解释大爆炸之后的宇宙的。这个大爆炸是否是飞面大神严重便秘后的一次严重拉稀，我们无从得知。

宇宙最终的结局，是热寂。仔细想想，如果宇宙是一场电影，那这个结局还真符合它的风格。

对于宇宙的目的、人生的意义这类话题，有时幻想作品中会给出有趣的解释。

*Rick And Morty* 中，Rick 发明了一个机器人。机器人问，我存在的意义是什么？Rick 回答道，递黄油。机器人沮丧地说，Oh my god！——我向他致敬，他的生命是有意义的。

《三体》中，归零者在全宇宙广播回归运动，呼吁让宇宙重启。但，宇宙又是从何而来？说到底归零者也不过是“刍狗”罢了。“归零者做的事，更像是一种宗教，一种行为艺术。”

《朝闻道》中，霍金问排险者，宇宙的目的是什么？排险者落寞地说，他怎么会知道呢。

《炎拳》中，阿格尼一次次地寻找人生的意义，一次次发现所谓的意义是虚假的。

《盲视》中，自我意识甚至是对智能的阻碍。人类的自我意识，也许只是生物演化的意外。

《搏击俱乐部》中，有人被拿枪指着头逼着去实现梦想。他是幸福的。

《蝴蝶效应》中，主角的每次穿越都会造成无法接受的结果，于是他穿越到还在母亲子宫内时自杀。
