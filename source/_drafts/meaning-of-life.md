title: 生命的意义
date: 2021-04-04 20:05:19
tags:
---

生命没有意义，“意义”只不过是人为定义的概念。

很久以前就得出这样的结论。如果在我有自我意识前可以选择，那么我会选择从未诞生。

后来才知道，这就叫虚无主义。

也问过同学，她似乎也有类似的观点，但又补充说，不妨把生命当作一个馈赠。

如果生命是一个程序，那么它或许类似于：

```
class Life {
    void findMeaningOfLife() {
        Life child;
        while ((child = new Life()) != null)
            child.findMeaningOfLife();
    }
}
```

这是一个无限的递归，不同的是，栈不会溢出。那么最初的调用者是谁？为什么要演化出生命？

据说宇宙产生于一次大爆炸，在那之前，没有意义。大爆炸是用来解释大爆炸之后的宇宙的。

宇宙最终的结局是热寂。

有时幻想作品中会给出解释。

Rick 发明了一个机器人，机器人问，我存在的意义是什么？Rick 回答道，递黄油。机器人沮丧地说，Oh my god！但他是有意义的。

《三体》中，归零者在全宇宙广播回归运动，呼吁让宇宙重启。但，为什么要让宇宙重启？宇宙最初从何而来？

《朝闻道》中，霍金问排险者，宇宙的目的是什么？排险者落寞地说，他怎么会知道呢。

《炎拳》中，阿格尼一次次地寻找人生的意义。

不用胡思乱想是一种幸福。
